ประเภทของเครื่องแม่ข่ายในระบบเครือข่ายคอมพิวเตอร์

1. Web Server (เว็บเซิร์ฟเวอร์)

หน้าที่: ให้บริการเว็บไซต์ผ่าน HTTP/HTTPS

ซอฟต์แวร์: Apache, Nginx, IIS

2. File Server (ไฟล์เซิร์ฟเวอร์)

หน้าที่: ให้บริการจัดเก็บและแบ่งปันไฟล์

ซอฟต์แวร์: Samba, Windows Server, TrueNAS

3. Authentication Server (เซิร์ฟเวอร์ยืนยันตัวตน)

หน้าที่: ตรวจสอบสิทธิ์การใช้งานของผู้ใช้

ซอฟต์แวร์: Active Directory, OpenLDAP, FreeIPA
